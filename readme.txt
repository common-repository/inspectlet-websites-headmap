This plugins allows us to add the script code from Inspectlet Free Plan

== Description ==

Allows to insert the script code (www.inspectlet.com - Free Plan) inside the <head> section to start recording our visitors activity inside our website.

Visit the <a href="http://www.seocom.es/">Seocom website</a> for more information about SEO or WPO optimization

== Installation ==

1. Install Inspectlet Websites HeadMap either via the WordPress.org plugin directory, or by uploading the files to your server inside the wp-content/plugins folder of your WordPress installation.
2. Activate Inspectlet Websites HeadMap plugin via WordPress Settings.
3. It's done. Easy, isn't it?
